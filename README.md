## ⚡ Overview
**Toxic Editor** is a professional-grade, high-performance web application designed for seamless PDF manipulation. Built with a sleek, neon-inspired "Glassmorphism" aesthetic, it offers a "Full App Experience" whether you are on a high-end PC or a modern smartphone.

Developed by **Aryan Jha (@jhaaryaan)**.

## 🚀 Key Features

### 🛠️ Core PDF Tools
- **Merge & Split**: Combine files or extract specific page ranges flawlessly.
- **Compress**: High-efficiency shrinking without losing visual clarity.
- **Security**: Professional-grade Password Protection and Encryption removal.
- **Rotate & Reorder**: Complete control over your document's orientation and page flow.

### 🪄 Advanced Magic
- **Image to PDF / PDF to Image**: Rapid format switching with batch processing.
- **Grayscale & Inversion (Dark Mode PDF)**: Render documents in high-contrast or B&W for better reading.
- **Flatten PDF**: Rasterize all layers to lock content and remove interactive elements.
- **Repair**: Recover content from structurally corrupted PDF files.

### 🛡️ Privacy & Clean Up
- **Redact Text**: Permanently black out specific words or sensitive information.
- **Strip Annotations**: Remove all comments, highlights, and note layers in one click.
- **Extract Images**: Rip raw embedded images out of any PDF into a ZIP folder.
- **Remove Blank Pages**: Auto-detect and strip empty pages from your scans.

## 🎨 Design Aesthetics
- **Toxic Neon Theme**: Vibrant greens and purples on a deep charcoal background.
- **Multi-Theme Support**: Instant toggle between high-energy Dark Mode and clean "Pro" Light Mode.
- **App GUI for Mobile**: Features a native-style 2-column grid and a fixed glass-blur bottom navigation bar.

## 🛠️ Tech Stack
- **Backend**: Python / Flask
- **Frontend**: Vanilla JS / CSS3 (Modern Flex/Grid)
- **Engines**: PyMuPDF (fitz), pikepdf, pypdf, reportlab, python-docx
- **Deployment**: Fully optimized for **Vercel**

## 📦 Local Installation

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd PDF
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python app.py
   ```

4. **Access in Browser**:
   Navigate to `http://127.0.0.1:5001`
